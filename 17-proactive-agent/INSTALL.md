# Install - proactive-agent

1. Create target folder:
   `mkdir -p ~/.openclaw/skills/proactive-agent`

2. Copy all files from this folder into target skill folder.

3. Verify these exist:
   - `~/.openclaw/skills/proactive-agent/SKILL.md`
   - `~/.openclaw/skills/proactive-agent/upstream-original/`

4. Restart/reload OpenClaw skills if needed.

5. Run a quick read test by asking the agent to summarize the skill usage from `upstream-original/SKILL.md`.
