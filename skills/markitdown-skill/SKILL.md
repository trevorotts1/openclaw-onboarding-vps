# markitdown-skill

Convert files (PDF, Word, Excel, PowerPoint, HTML, images, audio) to Markdown using Microsoft's markitdown tool.

## Install
Run: `pip install markitdown`

## Usage
```bash
markitdown input.pdf > output.md
markitdown input.docx > output.md
markitdown input.xlsx > output.md
```

## What it does
- PDF → Markdown
- Word (.docx) → Markdown  
- Excel (.xlsx) → Markdown
- PowerPoint (.pptx) → Markdown
- HTML → Markdown
- Images (with alt text) → Markdown
- Audio (transcription) → Markdown

## Notes
- Requires Python 3.8+
- Install: `pip install markitdown`
- Docs: https://github.com/microsoft/markitdown
