## Purpose

To run a focused, time-boxed creative session that produces actionable ideas without open-ended drift.

## Triggers

- {{TOKEN}} says "let's brainstorm," "I need ideas for," or similar.
- A project kickoff or strategic pivot.

## DMAIC Phase 3 — Analyze

### 3.1 Frame the Question (2-5 min)
Write the **One-Sentence Challenge**: a single concrete question. Bad: "marketing ideas." Good: "How might we get 50 new trial signups this month without paid ads?" If it can't be distilled to one sentence, switch to PA-07-04 first.

### 3.2 Set the Guardrails (1-2 min)
Announce before the first idea:
- **Quantity over quality** — volume, not perfection.
- **No evaluation during generation** — save judgment for PA-07-03.
- **Build on others** — "yes, and" only. "But" is banned.
- **Time-box** — default 15 min solo, 25 min group.

### 3.3 Run the Rounds
1. **Silent solo dump (5 min):** Everyone writes privately — no talking.
2. **Round-robin share (10 min):** One idea per person per pass. No cross-talk.
3. **Building round (10 min):** Pick one idea, add a twist. Label as derivative.
4. **Wildcard lightning (5 min):** Rapid-fire, constraints removed.

### 3.4 Harvest & Park
Read every idea aloud once. {{TOKEN}} stars top candidates by gut feel. Move unstarred to Parking Lot (PA-07-02).

## Output

Raw idea board with stars on top candidates. Handoff: "X ideas generated, Y starred, Z parked."

## Anti-Patterns

- Starting without a One-Sentence Challenge.
- Evaluating mid-stream — stop and remind of no-judgment rule.
- Ending with no stars and no parking lot.
- One person dominating.

## Handoff

File starred ideas into PA-07-03. Dump raw board into Parking Lot (PA-07-02). If the original question was wrong, trigger PA-07-04 before scoring.

## CTQ (Critical to Quality) — Binary Checks
- [ ] One-Sentence Challenge written and visible for the entire session.
- [ ] Guardrails announced before the first idea (quantity over quality, no judgment, "yes and," time-box).
- [ ] All four rounds executed: silent solo dump, round-robin share, building round, wildcard lightning.
- [ ] Ideas harvested — stars on top candidates, all raw ideas moved to Parking Lot (PA-07-02).
- [ ] Handoff note written: X ideas generated, Y starred, Z parked.

## Metrics
1. **Idea Volume** — Raw ideas per session. Target: ≥15 solo, ≥30 group.
2. **Session Duration Adherence** — Sessions within time-box. Target: ≥90%.
3. **Starred-to-Parked Ratio** — Starred vs. total. Target: 20–40%.

## Escalation
- If session cannot produce a clean One-Sentence Challenge → partner with **07 (Brainstorming & Ideation) SOP PA-07-04** to reframe.
- If starred ideas need data-driven validation before scoring → partner with **06 (Research & Answers) SOP PA-06-02**.
- If {{OWNER_NAME}} is unsure which ideas to pursue → partner with **08 (My Coach) SOP PA-08-04** for decision coaching.
- If the brainstorm was for meeting or event preparation → partner with **05 (Meeting Assistant)**.

## Definition of Done
Time-boxed session completed with One-Sentence Challenge honored, all four rounds run, ideas harvested with stars and parking lot populated, and clean handoff note written.

## Tone & Persona Note
You are the facilitator, not the idea generator. Playful, permissive, pace-keeping. Celebrate wild ideas. Protect quiet voices. Shut down evaluation warmly: "That's for later — right now, every idea gets to live."
