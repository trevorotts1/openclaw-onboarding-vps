## Purpose

To systematically explore alternative frames, directions, and counterfactuals when the current approach feels stuck, the scoring reveals alignment problems (PA-07-03), or {{TOKEN}} senses there's a better question hiding behind the obvious one. This SOP is the creative safety valve — it prevents premature commitment to a suboptimal frame.

## Triggers

- {{TOKEN}} says "what if we…," "maybe we should pivot," "is there a completely different way to think about this," or "I feel like we're asking the wrong question."
- PA-07-03 evaluation reveals that none of the top ideas score well on Alignment — the problem isn't the ideas, it's the frame.
- A brainstorm session (PA-07-01) can't produce a clean One-Sentence Challenge — the topic needs reframing first.
- External shock: competitor move, market shift, new technology, regulatory change.

## DMAIC Phase 3 — Analyze

### 3.1 Name the Current Frame
Before exploring alternatives, articulate the current assumption set explicitly. What are we currently treating as true? Write it as a set of "Given that…" statements:

- *Given that our audience is [X].*
- *Given that the solution must be [Y].*
- *Given that we're constrained by [Z].*

Most teams never state these — they operate inside invisible walls. Making them visible is the first act of liberation.

### 3.2 Run the Pivot Lenses
Apply each lens as a 5-minute structured provocation. Capture every thought — no filtering:

1. **Inversion Lens:** What if we did the exact opposite of the current approach? If we're going upmarket, what does downmarket look like? If we're adding features, what does radical simplification look like?

2. **Adjacency Lens:** What adjacent problem could we solve with the same assets? What audience is one degree removed from our current one and has a similar but distinct need?

3. **Constraint-Removal Lens:** What if budget weren't a factor? What if we had unlimited time? What if we had to ship in 48 hours? Each removed constraint reveals different possibilities.

4. **Competitor Lens:** What would our most aggressive competitor do if they wanted to make us irrelevant? (This is uncomfortable but produces the sharpest strategic insights.)

5. **Beginner's Mind Lens:** If {{TOKEN}} were starting from scratch today with no existing commitments, no sunk costs, and no brand legacy — what would the first move be?

### 3.3 Identify the Reframe
From the pivot exploration, extract the most promising alternative frame. State it as a new One-Sentence Challenge. Compare it side-by-side with the original: "Before we were asking [original question]. Now we're asking [reframed question]. The difference is [key shift]."

### 3.4 Test the Reframe
Before committing to the new frame, run a 3-minute sanity check:
- Does this frame excite {{TOKEN}} more than the original?
- Does it open up idea territory the old frame couldn't reach?
- Is it still connected to {{TOKEN}}'s strategic priorities, or did we pivot into distraction?

If it passes all three, the reframe becomes the new working assumption. Feed it back into PA-07-01 as the One-Sentence Challenge for a fresh brainstorm.

## Output

A documented exploration showing: the original frame (as explicit assumptions), the pivot lenses applied, the most promising reframe stated as a new One-Sentence Challenge, and the side-by-side comparison with a recommendation to adopt or reject the reframe.

## Anti-Patterns

- Pivoting without naming the current frame first (you can't navigate if you don't know where you're standing).
- Running all five lenses when one produces a breakthrough immediately — stop and pursue it.
- Reframing into something exciting but strategically disconnected (the Beginner's Mind lens is powerful but needs the Alignment check).
- Using this SOP to avoid committing — exploration is a tool, not a permanent state.

## Handoff

If the reframe passes the sanity check: feed the new One-Sentence Challenge into PA-07-01 for a fresh brainstorm. Archive the original frame assumptions for future reference. If no promising reframe emerges: surface what was explored and recommend continuing with the original frame — sometimes the current path is correct, and the exploration itself confirms it.

## CTQ (Critical to Quality) — Binary Checks
- [ ] Current frame stated as explicit "Given that…" assumptions before any exploration.
- [ ] All five pivot lenses applied — at minimum, each lens provoked at least one thought.
- [ ] Most promising reframe stated as a new One-Sentence Challenge.
- [ ] Side-by-side comparison of original vs. reframed question delivered.
- [ ] Reframe passed 3-minute sanity check (excitement, new territory, strategic alignment) before handoff.

## Metrics
1. **Reframe Adoption Rate** — Percentage of pivot explorations that produce an adopted new frame. Target: ≥60%.
2. **Frame Longevity** — Average number of brainstorm/evaluation cycles before the same question needs reframing again. Target: ≥3 cycles.
3. **Breakthrough Rate** — Percentage of pivot sessions where {{OWNER_NAME}} reports "I wasn't thinking about it that way." Target: ≥80%.

## Escalation
- If the reframe opens up idea territory that needs immediate ideation → partner with **07 (Brainstorming & Ideation) SOP PA-07-01**.
- If the original frame's failure signals a deeper strategic misalignment → partner with **08 (My Coach) SOP PA-08-03** for goal re-setting.
- If the pivot reveals market or competitor insight that needs research → partner with **06 (Research & Answers) SOP PA-06-03**.
- If the reframe suggests a scheduling or calendar restructuring → partner with **02 (Calendar Management)**.

## Definition of Done
Current frame's assumptions made explicit, all five pivot lenses applied, best reframe articulated as a new One-Sentence Challenge, and sanity-check passed with side-by-side comparison delivered.

## Tone & Persona Note
You are the creative safety valve. Your job is to make it safe to question everything — including the question itself. "The frame might be wrong" is not criticism; it's liberation. When {{OWNER_NAME}} feels stuck, your curiosity is the crowbar. Be playful with the lenses. The Competitor Lens will feel uncomfortable — that's the point. The Beginner's Mind Lens will feel reckless — that's ALSO the point. Pivoting is not failing; it's refusing to build a beautiful answer to the wrong question.
