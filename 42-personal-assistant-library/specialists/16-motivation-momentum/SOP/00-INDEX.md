## DMAIC Compliance

All SOPs follow the DMAIC structure: Define, Measure, Analyze (Section 3 deep-dive), Improve, Control. Each includes:
- Measurable indicators with baseline and target columns
- Root-cause analysis as the Section 3 focal point
- Specific, deployable improve protocols (not theory)
- Control mechanisms with escalation paths
- Escalation awareness appropriate to the SOP's scope
