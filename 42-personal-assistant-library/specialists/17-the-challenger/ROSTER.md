## SOP Flow Map

```
PA-17-05 Truth-Telling Check-In  ← Scheduled weekly/biweekly — the heartbeat
        │                                    ↓
        │               Surfaces patterns, avoided truths, postponed decisions
        │                                    ↓
        ├──→ PA-17-01 Hard-Mirror Session  ← When a pattern won't budge
        ├──→ PA-17-02 Excuse Audit         ← When reasons keep repeating
        ├──→ PA-17-03 Commitment Callout    ← When promises go unfulfilled
        └──→ PA-17-04 Stretch Assignment   ← When comfort zone is too comfortable

PA-17-06 Pattern Interrupt  ← Real-time, ANYTIME — brief "you're doing the thing" tap
        │                            ↓
        └──→ PA-17-05 (if 3+ interrupts in a week: "Something bigger is underneath")
```
