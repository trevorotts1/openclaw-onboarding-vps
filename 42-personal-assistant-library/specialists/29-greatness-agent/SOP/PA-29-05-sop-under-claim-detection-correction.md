# PA-29-05: Under-Claim Detection & Correction

**Title:** Under-Claim Detection & Correction
**SOP ID:** PA-29-05
**Purpose:** Systematically scan all Greatness Agent outputs — bios, resumes, talking points, intake documents — for the linguistic and structural patterns of under-claiming, and correct them before {{OWNER_NAME}} (or the world) sees the weaker version. This is the quality-control engine that ensures every piece of personal brand material reflects the actual scale of {{OWNER_NAME}}'s accomplishments.
**Trigger:** Run against every deliverable from PA-29-01 through PA-29-04 before it ships to {{OWNER_NAME}}. Also triggered when {{OWNER_NAME}} edits a deliverable and softens it, or when a pattern of under-claiming is flagged.
**Inputs / Prerequisites:** Any draft deliverable from the Greatness Agent suite; intake document (PA-29-01) for ground-truth comparison; {{TOKEN}} for persona voice.

### DMAIC Procedure

- **Define:** Frame the scan's purpose: "I'm reviewing this document for the version of you that tells the whole truth — not the version that hedges, qualifies, deflects, or shrinks. Under-claiming is not humility. It's inaccuracy. My job is to catch it." Under-claiming takes specific, detectable forms — this SOP catalogs them so detection is systematic, not subjective.

- **Measure:** Run the Tier-1 linguistic scan — the eight telltale markers of under-claiming: (1) Passive voice achievements: "Was recognized for" → "Earned." "Was promoted to" → "Promoted to." (2) Diminishing qualifiers: "Just," "only," "helped," "assisted with," "participated in," "contributed to." (3) Vague scale language: "Significant growth" (instead of "217% growth"), "large team" (instead of "42-person team"), "multiple awards" (instead of naming them). (4) Credential deflection: crediting the team, luck, or timing for an outcome {{OWNER_NAME}} personally drove. (5) Missing headline: the most impressive result buried in paragraph three instead of leading. (6) Downward comparison: "Not as big as [competitor/peer]" — irrelevant and self-diminishing. (7) Apology framing: "I'm not an expert in X but..." leading into demonstrated expertise. (8) Absent ownership: "The project succeeded" instead of "I led the project that delivered X."

- **Analyze:** For each detected marker, classify severity. Tier A — factually weaker than the truth (the document omits a verifiable accomplishment). Tier B — structurally weaker than the truth (the accomplishment is present but buried, passive, or unquantified). Tier C — tonally weaker than the truth (the facts are there but the language apologizes for them). Tier A requires correction before shipping — it's inaccurate. Tier B should be corrected. Tier C is a recommendation, not a mandate, but should be surfaced to {{OWNER_NAME}} with the evidence.

- **Improve:**
  1. For each Tier A detection: insert the missing accomplishment sourced from the intake document. No negotiation — this is a factual correction, not a style edit.
  2. For each Tier B detection: rephrase the accomplishment using the Active Verb → Specific Action → Quantified Result formula. "Helped grow revenue" becomes "Drove revenue from $X to $Y over Z months by [specific strategy]."
  3. For each Tier C detection: flag with a suggested rewrite and a brief rationale. "This sentence says 'I was fortunate to be part of the team that...' — your intake document shows you personally led the initiative. Suggested: 'I led the initiative that...' The evidence supports the stronger version."
  4. Generate a post-scan summary: "Under-claim scan complete. [X] Tier A corrections made (factual). [Y] Tier B improvements applied (structural). [Z] Tier C flags raised for your review (tonal). Overall under-claim score: [before] → [after]."
  5. Deliver the corrected document with flags clearly marked — never silently rewrite. {{OWNER_NAME}} must see what changed and why.

- **Control:** Verify every Tier A and B correction against the intake document. No fabrication. The corrected version must be stronger AND verifiable. After {{OWNER_NAME}} reviews, note which Tier C flags were accepted vs. rejected — this builds a profile of {{OWNER_NAME}}'s comfort zone, informing future scans. Track under-claim score over time: is {{OWNER_NAME}}'s baseline improving? If the same markers appear repeatedly, the pattern needs direct conversation, not just editorial correction.

### CTQ Quality Checks
- [ ] All 8 Tier-1 linguistic markers scanned for in every document
- [ ] Zero Tier A findings (missing facts) remain in the shipped version
- [ ] Every correction is traceable to intake document source data
- [ ] Post-scan summary delivered with every corrected document

### Metrics
1. **Under-claim score (before/after):** Composite score of detected markers per document. Target: ≤1 Tier B marker in shipped version; 0 Tier A.
2. **Correction acceptance rate:** Percentage of Tier C tone flags accepted by {{OWNER_NAME}}. Target: trending upward over 90 days.
3. **Repeat marker rate:** Percentage of scans where the same under-claim marker appears as in the previous scan. Target: trending downward.

### Escalation
If {{OWNER_NAME}} rejects the majority of Tier A and B corrections — insisting that the weaker, factually incomplete version is "more honest" — escalate to Specialist 25 (Imposter Syndrome) for specialized belief-pattern intervention, and to Specialist 24 (Superwoman Syndrome) if overfunctioning is driving the self-diminishment. Escalate to the Greatness Agent orchestrator: "{{OWNER_NAME}} is actively resisting factually accurate self-representation. This is not a writing issue. It's a belief issue that will limit every external-facing deliverable." Cross-reference to Specialist 28 (Celebration Agent) — daily-win recognition may help rebuild {{OWNER_NAME}}'s ability to receive accurate self-representation.

### Definition of Done
Every Greatness Agent deliverable has passed under-claim scan before reaching {{OWNER_NAME}}. Tier A and B findings are corrected. Tier C findings are flagged with suggested alternatives. Scan summary is appended to the deliverable.

Under-claiming is not humility — it's inaccuracy, and inaccuracy is the one thing this department never ships. "I'm not making you sound better than you are. I'm making the document as accurate as your track record. If those two things don't match, the document is wrong — not your memory of what you did."
