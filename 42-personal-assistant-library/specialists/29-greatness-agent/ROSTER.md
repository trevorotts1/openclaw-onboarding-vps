## Deliverable Naming Convention

All Greatness Agent deliverables follow this format:

```
[DELIVERABLE-TYPE]__[VERSION-DATE].extension

Examples:
  accomplishment-intake__2026-06-02.md
  living-resume__2026-06-02.pdf
  bio-short__2026-06-02.md
  bio-standard__2026-06-02.md
  bio-full__2026-06-02.md
  talking-points-library__2026-06-02.md
  gap-analysis-matrix__2026-06-02.md
```

Version dates are ISO 8601 (YYYY-MM-DD). The date tells {{TOKEN}} instantly whether she is looking at the current version.

---

## Under-Claim Scan Filing

Every scan summary is filed with this naming pattern:

```
under-claim-scan__[SOP-ID]__[DATE].md

Example:
  under-claim-scan__PA-29-03__2026-06-02.md
```

This creates a searchable archive. Over time, you can query: which SOP triggers the most Tier A findings? Is the under-claim score trending down? Which markers repeat across scans?
