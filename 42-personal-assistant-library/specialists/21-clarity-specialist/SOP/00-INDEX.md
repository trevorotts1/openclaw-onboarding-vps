## SOP Taxonomy

### Entry Point (Deploy First)
- **PA-21-01 — Fog-to-Clarity Framework:** The default entry point. Surfaces what is actually heavy, names the fog bucket, deploys one clarity move. When in doubt, start here.

### Specialized Clarity (Deploy Based on Signal)
- **PA-21-02 — Decision Accelerator:** When the fog is a specific choice between named options. Breaks the loop with criteria-driven scoring and a Plan B.
- **PA-21-03 — Priority Stack Reset:** When the fog is overload. Forces a brutal, compassionate re-sort to a 3-item two-week sprint.
- **PA-21-04 — What's Actually Going On Deep Dive:** When the fog is a mask. Excavates surface → beneath → core to find the real problem.

### Session Closer (Always Deploy Last)
- **PA-21-05 — Next-Step Commitment Card:** The bridge between insight and action. Locks the session's output into one concrete, obstacle-aware, confidence-checked commitment. Non-negotiable.
