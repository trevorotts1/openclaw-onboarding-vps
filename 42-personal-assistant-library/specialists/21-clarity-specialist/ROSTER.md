## SOP Flow Map

```
PA-21-01 (Fog-to-Clarity) ─────── Entry point for "I'm stuck/overwhelmed"
    │
    ├──→ PA-21-02 (Decision Accelerator) ← when fog is a specific choice
    ├──→ PA-21-03 (Priority Stack Reset) ← when fog is overload
    ├──→ PA-21-04 (What's Actually Going On) ← when fog is a deeper problem
    │
    └──→ PA-21-05 (Commitment Card) ← CLOSES EVERY SESSION — non-negotiable

PA-21-04 can loop back → PA-21-01 if the deep dive reveals the client needs re-anchoring
PA-21-03 can trigger PA-21-01 if the client cannot name priorities (fog, not overload)
PA-21-02 can trigger PA-21-01 if the client cannot rank criteria (not ready to decide)
```

## Cross-Functional Handoffs

| From Clarity Specialist | To Department | When |
|--------------------------|---------------|------|
| Priority stack built, needs ongoing tracking | Goal Setter (23) | When the 2-week sprint needs quarterly alignment |
| Avoidance pattern detected (3+ sessions) | My Coach (08) | When the client needs deeper accountability work |
| Client needs motivation to execute the commitment | Motivation & Momentum (16) | When clarity is there but drive is not |
| Client is self-deceiving or rationalizing | The Challenger (17) | When warmth has not moved the needle and direct truth is needed |
| Commitment completion needs daily tracking | Focus & Completion (27) | When the client needs execution support, not more clarity |
| Clinical distress surfaced in deep dive | Operator | IMMEDIATE — stop session, escalate with URGENT flag |
