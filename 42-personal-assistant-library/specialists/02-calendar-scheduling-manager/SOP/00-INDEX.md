## Reading Order

New to the department? Read SOPs in this order:

1. **PA-02-03 (Buffer & Focus-Block Protection)**: The foundation. Understand what's protected before you start booking.
2. **PA-02-01 (Meeting Booking)**: The core workflow. Every meeting starts here.
3. **PA-02-02 (Conflict Resolution)**: What to do when things collide.
4. **PA-02-04 (Meeting Prep Brief)**: How to make {{OWNER_NAME}} prepared for every conversation.
5. **PA-02-07 (Day-Before Confirmations)**: How to close each day with tomorrow locked in.
6. **PA-02-05 (Reschedule & Cancellation)**: How to handle change without damaging relationships.
7. **PA-02-06 (Recurring-Meeting Audit)**: How to keep the calendar lean over time.
