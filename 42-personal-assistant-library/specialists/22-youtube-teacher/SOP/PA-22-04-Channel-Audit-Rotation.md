# SOP PA-22-04 — Channel Audit & Rotation

**SOP ID:** PA-22-04
**Department:** 22 — YouTube Teacher / Curator
**Title:** Channel Audit & Rotation
**Version:** 1.0
**Last Updated:** 2026-06-02
**Estimated Time:** 20–40 minutes per audit

---

## Purpose

When {{TOKEN}}'s learning ecosystem needs maintenance — stale subscriptions, channels that have drifted, gaps new creators have filled — this SOP audits the active roster, removes dead weight, and rotates in fresh voices so the feed stays a learning engine, not a landfill.

---

## Section 3 — DMAIC Framework

### 3.1 Define

**Trigger:** Monthly calendar reminder fires, {{TOKEN}} says "refresh my channels," or a playlist consistently underperforms.

**What this SOP does:** Reviews every channel in {{TOKEN}}'s active learning roster. Retires channels that no longer serve. Discovers replacements. Produces a "keep, drop, add" report.

**What this SOP does NOT do:** Audit entertainment subscriptions. Delete without confirmation.

**Success looks like:** Every channel in the active roster directly supports a current learning goal. No zombie subscriptions.

### 3.2 Measure

| Metric | Target |
|--------|--------|
| Audit frequency | Monthly minimum |
| Channel coverage | 100% of active learning channels |
| Stale flag threshold | 0 watched videos in 60 days |
| Replacement quality | Every drop has a recommendation where a skill gap remains |

### 3.3 Analyze

**The Audit Path:**

1. **Inventory active channels** — pull from the subscription feed or master registry built in PA-22-01. Group by learning domain: sales, AI, leadership, personal development, industry-specific.

2. **Score each channel on 4 dimensions (1–5):**

   | Dimension | What to check |
   |-----------|---------------|
   | **Relevance** | Do recent videos still map to current learning goals? |
   | **Recency** | Uploaded in the last 90 days? |
   | **Signal Density** | Watch 2 min of 3 recent videos. Teaching or performing? |
   | **Watch History** | Has {{TOKEN}} watched this channel in the last 60 days? |

3. **Classify:** Keep (14+), Watch (10–13), Drop (below 10).

4. **For every dropped channel:** Did it cover a skill that still matters? Yes → find a replacement via PA-22-01's matching method. No → none needed.

5. **Discover new channels** where gaps exist or interests have shifted. Score candidates on the same 4 dimensions.

6. **Produce the report** — Keep (X channels by domain), Drop (Y channels, each with reason + replacement), Add (Z channels, each with one-line why).

7. **Present for approval** — never act unilaterally. "Monthly audit ready. 3 to drop, 2 to add. Proceed?"

### 3.4 Improve

- Same channel flagged borderline two months running → drop it. The pattern is the signal.
- New channel where {{TOKEN}} watches 5+ videos in week one → note what made it sticky and bias future discovery toward those qualities.
- Entire domain unwatched 60+ days → surface it. "4 leadership channels, none watched in 2 months. Still a priority?"

### 3.5 Control

- Never drop a channel without offering a replacement when the skill gap remains.
- Scores guide but do not dictate. A channel scoring 13 because of a 92-day gap but is the definitive source on its topic should survive with a note — not be auto-purged.
- {{TOKEN}} disagrees with a drop → channel stays. No debate. {{TOKEN}}'s preferences are final authority.
