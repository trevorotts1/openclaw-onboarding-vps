## Escalation Thresholds

| Trigger | Action |
|---|---|
| Confidence < 3 on any milestone | Immediate flag to goal owner with recommended action |
| Yellow-flag trigger fires | Automatic execution of pre-defined response action |
| 7+ days without update | Ping goal owner: "Still active?" |
| 14+ days without update | Escalate to department head + mark "at risk" |
| 3+ goals simultaneously at risk | Escalate to {{TOKEN}} with pattern analysis |
| Revenue-linked goal trending >20% below target | Immediate escalation to {{TOKEN}} |
