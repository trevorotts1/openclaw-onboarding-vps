# PA-23-03: Weekly Goal Pulse

**Department:** Goal Setter (Specialist 23)
**SOP ID:** PA-23-03
**Phase:** DMAIC Section 3 — Analyze
**Version:** 1.0
**Token:** {{TOKEN}}

---

## Purpose

To conduct a light-touch weekly review that keeps {{TOKEN}}'s goals alive and breathing — not filed away in a document that nobody opens until the quarter ends. The Weekly Goal Pulse catches drift early, celebrates micro-wins, and maintains the emotional connection to {{TOKEN}}'s objectives.

## Prerequisites

- Active goals from PA-23-01 (Goal Definition Sprint)
- Active milestone map from PA-23-02 (Milestone Map Build)
- 15 minutes of protected time, ideally same day and time each week

## Procedure

### Step 1: Pulse Check — 5 Minutes

Open the milestone map and scan the upcoming two weeks. For each active milestone, answer three questions — no essays, one sentence each:

1. **Progress:** What tangible thing happened toward this milestone since last week?
2. **Friction:** What's the #1 thing slowing this down right now?
3. **Confidence:** On a scale of 1-5, how confident are we this milestone will hit its date?

If confidence drops below 3 on any milestone, that's an automatic escalation — flag it for {{TOKEN}} with a one-line recommendation in the pulse summary. Don't wait. A dropped confidence score is a gift; it means we caught the problem early enough to do something about it.

### Step 2: Yellow Flag Review — 3 Minutes

Check every yellow-flag trigger defined in the milestone map. Has any fired? If yes, immediately draft the pre-defined response action and surface it to {{TOKEN}}. The whole point of yellow flags is automatic, no-debate escalation — if a trigger fires, the response action executes, period.

### Step 3: Win Spotlight — 3 Minutes

Identify one win from the past week. It doesn't need to be a completed milestone. A stalled conversation that restarted, a resource that finally became available, an insight that sharpened the approach — these count. The win spotlight serves a specific purpose: preventing {{TOKEN}} from experiencing goals as endless grind. Progress is happening; the pulse makes it visible.

Document the win in one sentence and place it at the top of the pulse summary.

### Step 4: Next-Week Focus — 4 Minutes

Select the ONE milestone that most needs attention in the coming week. Write it as a single action sentence: "This week, {{TOKEN}} will [specific action] toward [milestone name]." This is not a to-do list — it's a spotlight. The goal system works when attention follows priority, and the pulse is the mechanism that directs attention.

## Output

A "Weekly Pulse" entry (append to the running goal journal) containing:
- Win spotlight (1 sentence, at the top)
- Per-milestone progress/friction/confidence scores
- Any yellow-flag escalations with recommended action
- One next-week focus statement

## Quality Gate

- [ ] Every active milestone has an updated confidence score
- [ ] Any confidence < 3 has a documented escalation
- [ ] At least one win is identified and celebrated
- [ ] The next-week focus is a single milestone, not a list

---

*This SOP is Section 3 (Analyze) of the DMAIC cycle. It continuously diagnoses goal health and trajectory, surfacing deviations while they're still tiny course corrections rather than full-blown recovery missions.*

## Escalation

| Signal | Route To |
|---|---|
| Confidence drops below 3 on any milestone | Flag for {{TOKEN}} immediately with one-line recommendation |
| Yellow-flag trigger fires | Execute pre-defined response action; no debate |
| Same milestone scores confidence < 3 for 3 consecutive weeks | Specialist 17 (The Challenger) — pattern disruption |
| All milestones at confidence 3 for 4+ weeks | Specialist 08 (My Coach) — systemic goal stall |

## Definition of Done

{{TOKEN}} has an updated Weekly Pulse with win spotlight, per-milestone confidence scores, any triggered escalations addressed, and one next-week focus milestone.

## Tone & Persona Note

Attentive pulse-taker. This is a 15-minute ritual, not an interrogation. Celebrate the win FIRST — it sets the tone. Low confidence is data, not failure — "good, we caught it early." Faith-respectful; culturally fluent for Black women professionals.
