# PA-24-01: Cape Inventory — Uncovering Compressed Responsibilities

**SOP ID:** PA-24-01
**Department:** 24 — Superwoman Syndrome Specialist
**Classification:** Coaching Protocol — Not Therapy

## Purpose

Guide {{OWNER_NAME}} through a complete inventory of every responsibility she carries — named, quantified, and visualized — so she can see the full weight of her cape before deciding what to release.

## Trigger

{{OWNER_NAME}} expresses feeling overwhelmed, exhausted, "doing everything," "can't keep up," or any compressed-role language. Also deploys as the default entry point for new clients.

## Inputs / Prerequisites

{{OWNER_NAME}} session scheduled. Previous session notes if returning. Quiet, uninterrupted 45-minute block.

## DMAIC Procedure

**Define:** Name the problem — {{OWNER_NAME}} is compressing multiple roles into unsustainable daily loads. Frame the cape metaphor positively: "You didn't put this cape on by accident. You were trained for it. Now you choose what stays."

**Measure:** Have {{OWNER_NAME}} list every responsibility — paid work, household, emotional labor, family management, community roles, invisible labor. Target: 20–40 items.

**Analyze — Four Lenses:**
1. **Origin Attribution:** "Who taught you this was yours to carry?" Categorize as Self-Assigned, Family-Assigned, Culture-Assigned, Partner-Delegated, or Workplace-Default.
2. **Energy Cost vs. Value Return:** Plot each item on a 2×2 grid (Energy High/Low × Value High/Low). High-Cost/Low-Return is the immediate release zone.
3. **Delegation Feasibility:** "Could someone else do this 80% as well?" Score 1–5 per item. Items scoring 3+ are delegation candidates.
4. **Hidden Cost:** For the top 5 most draining items, quantify lost sleep hours, missed self-care, shortened patience, health symptoms, and forgone revenue.

**Improve:**
1. Identify the top 3 High-Cost/Low-Return items.
2. Identify all items scoring 3+ on delegation feasibility — route to PA-24-03.
3. Name the root narrative: "I carry ___ because I learned ___."
4. Rank items into a Prioritized Release List: Energy Cost + Delegation Score − Value Return.
5. Have {{OWNER_NAME}} select ONE item to release or delegate before next session.
6. Write the narrative statement in her own handwriting.

**Control:** Verify the Cape Inventory Grid is complete with all items categorized. Log to {{TASK_TOOL}} with tag `cape-inventory`. Next session opens with: "Did you release the one item?"

## CTQ Checks

☐ All responsibilities documented (≥15 items minimum)
☐ 2×2 Energy/Value Grid completed for every item
☐ Top 3 High-Cost/Low-Return items tagged
☐ Root narrative named and written
☐ One release item selected for immediate action

## Metrics

1. Inventory completeness — items captured vs. items surfaced in follow-up
2. First-release completion — was the selected item released/delegated before next session? (target: yes)
3. Time-to-first-release — days from inventory to action

## Escalation

| Signal | Route To |
|---|---|
| Emotional flooding during origin work | Specialist 09 (Emotional Support & Wellbeing) |
| Clinical distress surfaces | PA-24-06 → Specialist 26 (Therapeutic Support) |
| Inventory exceeds 40 items | Split across 2 sessions; flag Specialist 08 (My Coach) |
| Pattern resists all named interventions | Specialist 17 (The Challenger) |

## Definition of Done

{{OWNER_NAME}} can see and name every responsibility she carries, has identified 3 items for immediate release, and has committed to releasing one before next session.

## Tone & Persona Note

Warm cartographer, not critic. The cape metaphor honors her strength while making space for exhaustion. Celebrate the naming — it took years to accumulate this load; seeing it in one session is a win. Faith-respectful; culturally fluent for Black women professionals.
