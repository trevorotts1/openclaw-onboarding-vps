## 1. Purpose
Conduct a structured audit of imposter thoughts — identifying distortion patterns, quantifying frequency and impact, and installing cognitive reframes that weaken the imposter cycle over time.

## 2. Scope Boundary
⚠️ **COACHING ONLY — NOT THERAPY.** Uses coaching-grade cognitive restructuring (thought identification, pattern recognition, reframing). Danger signals: ego-dystonic intrusive obsessions, inability to distinguish thought from reality, compulsive rituals → STOP and warm-refer per PA-25-06.

## 3. Procedure (DMAIC)

### Define
Capture 5–10 imposter thoughts over 3 days using a thought log. Categorize by imposter type: Perfectionist, Natural Genius, Soloist, Expert, Superperson. Identify the triggering situation for each.

### Measure
Rate each thought: Frequency (daily/weekly/monthly), Intensity (1–10), Impact (avoidance, over-preparation?). Calculate "Imposter Load Score": average intensity × frequency modifier. Track challenged vs. accepted thoughts.

### Analyze
Map each thought to its cognitive distortion: catastrophizing, mind-reading, discounting positives, all-or-nothing, labeling, personalization, fortune-telling. Identify the root belief ("I am not qualified," "They'll find out"). Pinpoint earliest career memory where this pattern appeared.

### Improve
- For each distortion, install a specific reframe: "They'll find out I don't know anything" → "I was hired because of my track record, and I ask smart questions."
- Teach the "3-Second Audit": when an imposter thought fires, the client asks: (1) Is it true? (2) Is there counter-evidence? (3) What would I tell a friend with this thought?
- Practice reframes aloud in session — the client must HEAR themselves say the counter-statement.
- Combine with PA-25-01 Evidence Ledger: each reframe links to at least one ledger entry.

### Control
- Weekly thought log review (first 4 weeks, then biweekly).
- Install a "reframe alarm" — a daily calendar reminder to review the top 3 reframes.
- Escalation trigger: if Imposter Load Score increases 2+ consecutive weeks despite full protocol compliance, reassess for clinical referral per PA-25-06.

## 4. Deliverables
Completed 3-Day Thought Log, Distortion-to-Reframe Map (one per distortion pattern), Imposter Load Score baseline + weekly tracker.

## 5. Integration
Feeds PA-25-01 (Evidence Ledger) — each reframe links to concrete evidence. Prep step for PA-25-03 (Owning the Room). Overlaps with PA-25-04 (Credentials-to-Confidence).

## 6. Outcome Metrics
Imposter Load Score decreases ≥30% within 6 weeks. Client articulates top 3 patterns and reframes without notes. ≥50% of imposter thoughts challenged independently by week 4.
