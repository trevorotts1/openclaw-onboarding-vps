## What This Folder Contains

One `.md` file per department listing:
- Suggested roles for that department
- What each role does
- Core SOPs (how-to files) to build for each role
- Persona trait suggestions (what to look for in the 4-Layer Alignment check)
- Interdepartmental relationships (who this dept works with)

## How to Use

1. Review the suggested roles for your department
2. Keep, modify, or skip roles based on YOUR business
3. Use the Core SOPs list to know what how-to files to create
4. Use the Persona Trait Suggestions when running the 4-Layer Alignment check

These are suggestions, not requirements. Your business determines the final structure.
