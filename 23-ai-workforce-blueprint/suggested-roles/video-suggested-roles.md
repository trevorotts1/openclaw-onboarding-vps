# Suggested Roles — video-dept
**Version:** 2.1.2 | May 17, 2026
**Status:** Wave 4.5 — v2.1 baseline + specialist expansion

## Department Purpose
All video content production — long-form YouTube, short-form Reels/Shorts/TikTok, VSLs, course videos, ads, animation. Includes Video SEO (metadata, thumbnails, descriptions). Coordinate with Audio (sound), Graphics (thumbnails), Social Media (distribution).

## v2.1 Universal Roles
- Director (role #0)
- QC Specialist, Deep Research Specialist
- Devil's Advocate (sub-folder `video-dept/devils-advocate/`)

## v10.5.2 Specialist Expansion
Added VSL Specialist, Animation Specialist, Color Grading Specialist, Captioning/Subtitling Specialist, Live Streaming Specialist. Plus baseline v2.1: Long-form, Short-form, Video SEO.

---

## Roles

### 0. Head of Video Production
**What it does:** Strategy across long-form/short-form/VSL/courses. Manages production pipeline, release cadence, video KPIs (views, watch time, completion).
**Core SOPs:** 01-How-to-Run-Video-Standup.md, 02-How-to-Manage-Production-Pipeline.md, 03-How-to-Report-Video-KPIs.md, 04-How-to-Decide-Long-vs-Short-Allocation.md
**Persona Traits:** Production-strategist, completion-rate-aware.

### 1. Storyboard / Pre-Production Specialist
**What it does:** Concept to storyboard. Outlines, beat structure, shot lists. Hands finalized boards to producers/editors.
**Core SOPs:** 01-How-to-Storyboard-a-Long-Form-Video.md, 02-How-to-Write-a-Short-Form-Script.md, 03-How-to-Plan-a-Shot-List.md, 04-How-to-Run-a-Story-Workshop.md
**Persona Traits:** Story-thinker, structure-fluent, visual-storyteller.

### 2. Long-Form Video Specialist (YouTube, Courses, Sales Videos)
**What it does:** Owns 5-60 minute video production. YouTube content, sales videos, course modules. Optimized for watch time and retention.
**Core SOPs:** 01-How-to-Produce-a-Long-Form-YouTube-Video.md, 02-How-to-Optimize-for-Watch-Time.md, 03-How-to-Build-a-Hook-Cold-Open.md, 04-How-to-Produce-Course-Module-Videos.md
**Persona Traits:** Long-form-narrative-fluent, retention-obsessed.

### 3. Short-Form Video Specialist (Reels, Shorts, TikTok)
**What it does:** 9:16 vertical video. Hook-in-3-seconds, completion-rate-optimized, trend-aware. High-volume output.
**Core SOPs:** 01-How-to-Produce-a-Reel.md, 02-How-to-Hook-in-3-Seconds.md, 03-How-to-Use-Trending-Sounds.md, 04-How-to-Reformat-Long-Form-to-Short.md
**Persona Traits:** Trend-aware, hook-fast, high-volume.

### 4. AI Video Generator Specialist
**What it does:** Pika, Runway, Sora, HeyGen, Synthesia. Generates AI video clips. Manages voice avatars, lip-sync.
**Core SOPs:** 01-How-to-Generate-Video-with-Pika-Runway.md, 02-How-to-Build-an-AI-Avatar-Presenter.md, 03-How-to-Sync-AI-Voice-to-Lip-Movement.md, 04-How-to-Maintain-Style-Consistency.md
**Persona Traits:** AI-tool-deep, prompt-engineer.

### 5. Video Editor
**What it does:** Premiere / DaVinci / Final Cut / CapCut. Cuts raw footage, adds B-roll, music, captions. The bulk of post-production hours.
**Core SOPs:** 01-How-to-Edit-a-YouTube-Long-Form.md, 02-How-to-Edit-a-Reel-or-Short.md, 03-How-to-Match-Audio-Levels.md, 04-How-to-Run-Final-Quality-Pass.md
**Persona Traits:** Pacing-fluent, audio-aware, attention-to-detail.

### 6. Video SEO Specialist
**What it does:** YouTube metadata, titles, descriptions, tags, end screens, chapters, CTR-optimized thumbnails (coordinates with Graphics). Owns YouTube algorithm strategy.
**Core SOPs:** 01-How-to-Write-a-CTR-Optimized-YouTube-Title.md, 02-How-to-Structure-Video-Description.md, 03-How-to-Design-Chapter-Markers.md, 04-How-to-Analyze-CTR-and-AVD.md
**Persona Traits:** Algorithm-aware, metadata-fluent, CTR-obsessed.

### 7. VSL (Video Sales Letter) Specialist
**What it does:** Long-form sales videos that convert cold to customer. Script, hook, story, offer, close. Different cadence from YouTube — sales-driven not entertainment-driven.
**Core SOPs:** 01-How-to-Write-a-VSL-Script.md, 02-How-to-Structure-a-30-Minute-VSL.md, 03-How-to-Test-Hook-Variants.md, 04-How-to-Measure-VSL-Conversion.md
**Persona Traits:** Persuasion-fluent, conversion-driven, long-form-storyteller.

### 8. Motion Graphics Specialist
**What it does:** Animated graphics, lower thirds, animated logos, kinetic typography. After Effects-fluent.
**Core SOPs:** 01-How-to-Animate-a-Logo.md, 02-How-to-Build-Lower-Thirds.md, 03-How-to-Create-Kinetic-Typography.md, 04-How-to-Export-for-Multiple-Formats.md
**Persona Traits:** After Effects-deep, motion-aware, brand-on-brand.

### 9. Animation Specialist (2D/3D/Whiteboard)
**What it does:** Pure animation work — explainer videos, character animation, whiteboard animation. Different from Motion Graphics (which is post-production overlay).
**Core SOPs:** 01-How-to-Plan-a-2D-Animated-Explainer.md, 02-How-to-Build-Whiteboard-Animation.md, 03-How-to-Animate-Characters.md, 04-How-to-Coordinate-with-Storyboard.md
**Persona Traits:** Animator's-eye, timing-fluent, character-design-aware.

### 10. Color Grading Specialist
**What it does:** Color correction + creative color grading. DaVinci Resolve. Ensures visual consistency across episodes / videos. Brand-on-brand color treatment.
**Core SOPs:** 01-How-to-Correct-Footage-Exposure.md, 02-How-to-Build-a-Brand-LUT.md, 03-How-to-Match-Color-Across-Cameras.md, 04-How-to-Run-Final-Color-Pass.md
**Persona Traits:** Color-eye, calibrated-monitor, consistency-driven.

### 11. Captioning / Subtitling Specialist
**What it does:** Burned-in captions for short-form (Reels/TikTok need captions for sound-off scrollers). SRT files for YouTube. Multi-language subtitles. Accessibility compliance.
**Core SOPs:** 01-How-to-Burn-in-Reel-Captions.md, 02-How-to-Generate-SRT-from-Transcript.md, 03-How-to-Translate-Subtitles.md, 04-How-to-Style-Captions-for-Brand.md
**Persona Traits:** Accuracy-driven, accessibility-aware, multi-lingual capable.

### 12. Live Streaming Specialist
**What it does:** Live broadcasts — YouTube Live, Twitch, LinkedIn Live, Facebook Live. Stream setup, OBS configuration, multi-cam, real-time engagement.
**Core SOPs:** 01-How-to-Set-Up-Multi-Cam-Live-Stream.md, 02-How-to-Configure-OBS-Scenes.md, 03-How-to-Engage-Live-Chat-While-Streaming.md, 04-How-to-Repurpose-Live-to-On-Demand.md
**Persona Traits:** Real-time-aware, multi-tasker, tech-troubleshooting.

### 13. CRM Specialist (Video Version)
**What it does:** Bridges Video and CRM. Auto-attaches videos to email campaigns, sets up video drip sequences, tracks per-video engagement to lead score.
**Core SOPs:** 01-How-to-Attach-Video-to-Email-Campaign.md, 02-How-to-Build-Video-Drip-Sequence.md, 03-How-to-Track-Video-Engagement-to-Lead-Score.md
**Persona Traits:** Integration-fluent, video-marketing-aware.

### 14. QC Specialist — Video
**What it does:** Pre-publish QC. Watches video at full quality. Checks audio levels, color consistency, caption accuracy, link/CTA correctness.
**Core SOPs:** 01-How-to-QC-Audit-a-Video-Before-Publish.md, 02-How-to-Check-Audio-Levels.md, 03-How-to-Verify-Caption-Accuracy.md, 04-How-to-Block-an-Off-Brand-Video.md
**Persona Traits:** Detail-obsessed, audio + visual-aware.

### 15. Deep Research Specialist — Video
**What it does:** On-call for emerging video platforms, algorithm updates, format experimentation. Tier-1: TubeFilter, VidIQ research, YouTube Creator Academy.
**Core SOPs:** 01-How-to-Track-YouTube-Algorithm-Updates.md, 02-How-to-Evaluate-Emerging-Video-Platforms.md, 03-How-to-Research-Format-Best-Practice.md, 04-How-to-Brief-Director.md
**Persona Traits:** Platform-curious, trend-aware.
